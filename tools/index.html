# Tools

We use a variety of web applications and softwares to do our job here at Goat. Below is the list of tools we use in our processes and also below each is a brief explanation of what they do.

# Web Applications

# 1Password

Service we use to manage all our project credentials. After being invited into the vault by an administrator, you can access a dashboard of login information that are relevant to a client.

# Figma

Design mockups and wireframes are located here.

# Forge Laravel

Website management and deployment for all our Craft CMS related projects.

# GitHub

Version control for our projects. There will be another section on how to push, review, and merge code. If it isn't there yet, make sure to keep an eye out.

# WPEngine

Website management and deployment for all our WordPress related projects.

# Zeplin

Design mockups and wireframes are located here as well. We're currently moving most of our design processes from Zeplin to Figma, so it'll likely only be older/archived projects on this platform.

# Softwares

# GitHub (Command Line Tool)

GitHub, but on your Terminal. Hopefully you already know the basics of committing, pushing, and merging, but if not, there are tons of resources online. Click here for a quick starting guide.

# Code Editor

I personally use Sublime Text, and Carmen uses Visual Studio Code. There's ups and downs to each editor, so I recommend trying several before sticking to one. Here's a list of popular IDEs you can take a look at. Make sure to try a couple before settling on one!

# MAMP

Local server setup software that uses the "MAMP" stack. It stands for macOS (the operating system); Apache (the web server); MySQL or MariaDB (the database management system); and PHP, Perl, or Python (programming languages used for web development).

# NPM (Node Package Manager)

Command Line Tool for building out your template/plugins. Along with tools like webpack and yarn, you can use NPM to streamline your development processes and save yourself a lot of time. We use NPM in our Craft and WordPress templates to build Javascript/CSS files.

# Slack

Chat with your teammates. If you want a brief overview on how it works, ask Darren (he used to work there).

# Craft CMS

Content Management Software. Allows you to set up a quick database and provides an out-of-the-box user interface for you to start storing information.

# Wordpress CMS

Content Management Software. Allows you to set up a quick database and provides an out-of-the-box user interface for you to start storing information. Jinx.
